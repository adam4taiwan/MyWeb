# 每日運勢判斷規則文件

## 系統架構概覽

每日運勢分四層加成，由上至下疊加輸出：

```
Layer 1：日柱基礎（日干 + 日支，通用，來自 KB）
Layer 2：命主加成（日主 + 今日十神 + 身強弱判斷，來自 KB）
Layer 3：大運加成（當前大運十神，來自 KB）
Layer 4：紫微命宮加成（命宮主星，來自 KB）
```

無命盤用戶：只有 Layer 1（呼叫 Gemini `/Fortune/daily`）
有命盤用戶：四層全套 KB 輸出（`/Fortune/daily-personal`，無 Gemini）

---

## Layer 1：日柱基礎

### 資料來源
- KB 表：`FortuneRules`，`Subcategory = "日干"` 或 `"日支"`
- 注入資料：`daily_fortune_rules.csv`（132 筆，日干×10 + 日支×12，各 6 個領域）

### 計算方式
1. 以 1900/1/31（甲子日）為基準，計算今日距離天數
2. `天干 index = days % 10`，`地支 index = days % 12`

### 輸出欄位
| 欄位 | 來源 |
|------|------|
| 總評 | 日干總評 + 日支總評（去重合併）|
| 財運/事業/感情/健康 | 日干規則為主 |
| 今日提醒 | 日支規則為主，日干補充 |

## Layer 2：命主加成

### 前置：日主計算
- 日主 = 出生日期的日柱天干（同上述 `GetGanZhi` 方法）

### 前置：今日十神計算
依日主天干 vs 今日天干的五行關係判定：

| 關係 | 陰陽相同 | 陰陽相異 |
|------|---------|---------|
| 同五行 | 比肩 | 劫財 |
| 日主生對方 | 食神 | 傷官 |
| 日主剋對方 | 偏財 | 正財 |
| 對方剋日主 | 七殺 | 正官 |
| 對方生日主 | 偏印 | 正印 |

五行相生：木→火→土→金→水→木
五行相剋：木剋土、土剋水、水剋火、火剋金、金剋木

### 身強弱評分

#### 月令（最高權重）
月支藏干查表，對日主有幫助的天干計分：

| 月支 | 藏干（權重）|
|------|------------|
| 子 | 癸(8) |
| 丑 | 己(5) 癸(2) 辛(1) |
| 寅 | 甲(5) 丙(2) 戊(1) |
| 卯 | 乙(8) |
| 辰 | 戊(5) 乙(2) 癸(1) |
| 巳 | 丙(5) 戊(2) 庚(1) |
| 午 | 丁(5) 己(3) |
| 未 | 己(5) 丁(2) 乙(1) |
| 申 | 庚(5) 壬(2) 戊(1) |
| 酉 | 辛(8) |
| 戌 | 戊(5) 辛(2) 丁(1) |
| 亥 | 壬(5) 甲(3) |

計分規則（僅計幫身五行 = 比劫同五行 or 印星生我五行）：
- 比劫（同五行）：權重 ≥5 → +3，權重 2-3 → +2，權重 1 → +1
- 印星（生我五行）：權重 ≥5 → +2，其他 → +1

#### 天干幫身（年、月、時柱，不含日干）
- 每個比劫或印星天干 → +1

#### 地支通根（四柱全部地支）
- 每個地支的藏干含比劫或印星五行 → +1

#### 地支刑沖扣分（命局內部）
| 情況 | 扣分 |
|------|------|
| 日支被六沖 | -2 |
| 月支被六沖（且不含日支） | -1 |
| 三刑（寅巳申 或 丑戌未，至少兩支同時存在） | -1 |

六沖對照：子午、丑未、寅申、卯酉、辰戌、巳亥

#### 身強弱判定
| 評分 | 判定 | 說明 |
|------|------|------|
| ≥ 5 | 身強 | 日主旺，需洩耗 |
| 3–4 | 中和 | 難以明確判定 |
| ≤ 2 | 身弱 | 日主弱，需扶助 |

### 今日十神順/逆勢判斷

| 身強弱 | 今日十神類型 | 顯示 |
|--------|------------|------|
| 身強 | 食傷財官殺（洩耗剋） | **順勢** |
| 身強 | 比劫印（幫生） | **逆勢** |
| 身弱 | 比劫印（幫生） | **順勢** |
| 身弱 | 食傷財官殺（洩耗剋） | **逆勢** |
| 中和 | 任何 | 順逆皆顯示 |

十神分類：
- 食傷財官殺：食神、傷官、偏財、正財、七殺、正官
- 比劫印：比肩、劫財、偏印、正印

### 刑沖會合破害（今日日支 vs 命局四柱地支）

今日日支與命局任一地支發生以下關係，附加一行說明：

| 關係 | 順勢時影響 | 逆勢時影響 |
|------|-----------|-----------|
| 六沖 | 能量受阻，好事多折，宜謹慎行事 | 壓力加劇，宜低調避衝 |
| 六合 | 機遇受合化，方向可能轉變 | 合化有緩解，壓力稍降 |
| 六刑 | 易生波折，計劃易被打斷 | 耗損加重，小心意外損傷 |
| 六破 | 成果易損，防虎頭蛇尾 | 情況複雜，謹防小人 |
| 六害 | 暗中阻滯，防無心之失 | 小人干擾，謹言慎行 |

六合：子丑、寅亥、卯戌、辰酉、巳申、午未
六刑：子卯、寅巳、寅申、申亥、丑戌、丑未（部分）
六破：子酉、丑辰、午辰、寅申、巳亥
六害：子未、丑午、寅巳、卯辰、申亥、酉戌

## Layer 3：大運加成

### 資料來源
- ChartJson 中的 `baziLuckCycles` 陣列
- 每筆：`startAge`、`endAge`、`liuShen`（十神縮寫）、`heavenlyStem`、`earthlyBranch`

### 計算方式
1. 當前歲數 = 今年 - 出生年
2. 找 `startAge ≤ 歲數 ≤ endAge` 的大運區間
3. 十神縮寫展開為全名（梟→偏印、殺→七殺 等）

### 縮寫對照表
| 縮寫 | 全名 |
|------|------|
| 比 | 比肩 |
| 劫 | 劫財 |
| 食 | 食神 |
| 傷 | 傷官 |
| 財 | 偏財 |
| 才 | 正財 |
| 殺 | 七殺 |
| 官 | 正官 |
| 梟 | 偏印 |
| 印 | 正印 |

### 輸出
- 查 KB `十神應事` 規則，喜用 → 大運順勢，忌用 → 大運逆勢
- 目前不過濾（順逆皆顯示），未來可依身強弱過濾

## Layer 4：紫微命宮加成

### 資料來源
- `UserCharts.MingGongMainStars`（命宮主星，逗號分隔，如 `天相` 或 `紫微,天府`）
- KB 表：`Subcategory = "命宮主星"`，14 顆主星各一筆規則
- 注入資料：`ziwei_minggong_rules.csv`（14 筆）

### 計算方式
- 從紫微斗數命盤（ChartJson palaces）找命宮宮位的主星
- 星曜縮寫展開（巨→巨門、殺→七殺 等）

## 快取機制

| 端點 | 快取 Key | 說明 |
|------|---------|------|
| `/Fortune/daily-kb` | `UserId` | 通用版，按用戶快取 |
| `/Fortune/daily-personal` | `personal:{UserId}` | 個人化版，獨立 key 避免衝突 |
| `/Fortune/daily` | `UserId`（有命盤則走 personal） | 統一入口 |

快取欄位：`DailyFortunes` 資料表，`FortuneDate = UTC今日`

清除方式：
- 用戶自清：`DELETE /Fortune/my-cache-today`（進入會員中心時自動呼叫）
- Admin 全清：`DELETE /Admin/fortune-cache-today`

---

## 端點對照

| 端點 | 用途 | Gemini |
|------|------|--------|
| `GET /Fortune/daily` | 統一入口：有命盤→KB，無命盤→Gemini | 條件式 |
| `GET /Fortune/daily-personal` | KB 四層個人化（需生辰+命盤） | 否 |
| `GET /Fortune/daily-kb` | KB 通用版（只需登入） | 否 |

---

## 前端路由邏輯（member/page.tsx）

```
進入會員中心
  └→ DELETE /Fortune/my-cache-today（清今日快取）
  └→ GET /Astrology/my-chart（查命盤是否存在）
       ├→ 有命盤 → GET /Fortune/daily-personal
       └→ 無命盤 → GET /Fortune/daily（Gemini）
```

無命盤時顯示橘色提示框，引導用戶前往排盤工具填寫生辰。
