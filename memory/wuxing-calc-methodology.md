# 五行計分方法論

## 程式碼方法（LfCalcWuXingMatrix）

- 天干各 10分 × 季節乘數
- 月支 30分（其他地支 10分）× 藏干比例 × 季節乘數 × 地支關係乘數（三合/沖/合等）
- 季節乘數範圍：木春1.8、土春0.2，極差9倍

## 玉洞子傳統手算方法

- 8字 = 8分固定，每字1分
- 地支按藏干比例拆分（如寅：甲×0.5, 丙×0.3, 戊×0.2）
- 月令旺相只作標記，不影響分值
- 近遠、旺相由命理師主觀加減判斷，非數值化
- 結果直觀，可手算驗證

## 兩種方法差異對照（1987/02/23 癸日，丁卯壬寅癸卯癸亥）

| | 程式碼 | 傳統手算 |
|--|--|--|
| 水(比印) | 29% | 46% |
| 木 | 52% | 35% |
| 火 | 18.5% | 16% |
| 土 | 0.8% | 2.5% |

主要差距來源：程式碼的月支30分×1.8讓木一枝獨秀，水被稀釋。
火土比例兩種方法差不多，差距集中在木水對調。

## 月支藏干分散係數（2026-04-10 加入）

### 問題

月支3藏干時，主氣（如寅=甲）雖只拿50%比例，但仍享有完整1.8倍季節乘數，
相比1藏干月支（子=癸全拿1.0×1.8）並不公平——能量分散的月支不應等同集中月支的旺氣。

### 實作

在 `LfCalcWuXingMatrix` 的月支 `AddBranch` 加入 `dispersionFactor`：

| 月支藏干數 | 代表月支 | 分散係數 |
|--|--|--|
| 1個 | 子、卯、酉 | 1.0（不折扣） |
| 2個 | 亥、午 | 0.85 |
| 3個 | 寅、申、巳、辰、戌、丑、未 | 0.75 |

```csharp
int mHiddenCount = LfBranchHiddenRatio.TryGetValue(mBranch, out var mHid) ? mHid.Count : 1;
double mDispersion = mHiddenCount >= 3 ? 0.75 : mHiddenCount == 2 ? 0.85 : 1.0;
AddBranch(mBranch, 30, mDispersion);
```

### 效果（同案例）

寅月主氣木：30 × 0.5 × 1.8 × 0.75 × 0.825 = 16.7（原22.3）
→ 木約47%（原52%），水約32%（原29%），更接近傳統手算感覺

## 身強弱分級臨界值（LfGetBodyStrengthLabel）

| 比印陣% | 用詞 |
|---------|------|
| ≥ 70% | 身強（極強） |
| 60~69% | 身強 |
| 45~59% | 中和 |
| 30~44% | 身弱 |
| < 30% | 身弱（極弱） |

臨界值由玉洞子確認（2026-04-10）：< 30 才算極弱，30~44 是身弱有根氣。
