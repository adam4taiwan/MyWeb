## 二、計算方法

### 地支順序（固定）
子(0) 丑(1) 寅(2) 卯(3) 辰(4) 巳(5) 午(6) 未(7) 申(8) 酉(9) 戌(10) 亥(11)

### 計算公式
```
神煞位置 = (流年地支索引 - 起算地支索引 + 12) % 12 + 1
```
- 位置 1 = 太歲，位置 2 = 太陽，... 位置 12 = 病符

## 三、兩種情境（同一年份，兩套起算）

### 情境一：以「生肖（年支）」起算 —— 外在環境對你的衝擊
- **用途**：看外在大環境如何影響你，也是民俗安太歲的依據
- **起算點**：命主出生年支（生肖）
- **範例**：生肖屬戌（狗），2026 丙午年
  - 起算：戌(10)，流年：午(6)
  - 位置 = (6 - 10 + 12) % 12 + 1 = 9 → **白虎（飛廉星）**
  - 含義：外在有血光之象（如生產、手術、意外），需祭白虎

### 情境二：以「命宮地支」起算 —— 個人內在運勢與心境
- **用途**：看個人真實遭遇與身心狀態，準確度高於生肖起算
- **起算點**：紫微命宮地支
- **範例**：命宮在丑，2026 丙午年
  - 起算：丑(1)，流年：午(6)
  - 位置 = (6 - 1 + 12) % 12 + 1 = 6 → **死符/小耗星**
  - 含義：個人財物體力耗損，產後需調養，支出增加

### 兩者衝突時的取捨規則
- **以命宮起算為準**看個人吉凶（命宮比生肖更個人化）
- **以生肖起算為準**看民俗趨吉避凶（如安太歲、補運）

## 七、程式實作規則

### 計算函數
```csharp
// 計算十二歲君位置（1-12）
int CalcSuiJunPos(string targetBranch, string startBranch) {
    string[] order = {"子","丑","寅","卯","辰","巳","午","未","申","酉","戌","亥"};
    int tIdx = Array.IndexOf(order, targetBranch);
    int sIdx = Array.IndexOf(order, startBranch);
    return (tIdx - sIdx + 12) % 12 + 1;
}
```

## 注意事項

1. **生肖起算 vs 命宮起算**：兩者用同一組十二歲君，但起算點不同，結果不同
2. **命宮起算更精準**：因為命宮具有個人化，全球同生肖者達 1/12，但命宮相同者極少
3. **不可混用**：流年十二神煞（順數）≠ 小限行年（男順女逆）
4. **無紫微命盤時**：僅做生肖起算，命宮起算標注「需紫微命盤」
