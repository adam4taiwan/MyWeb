# 八字方位風水・出生環境章節

## 概念說明

八字方點陣圖：年月日時四柱各代表一個方位，揭示命主懷胎時（父母受孕之地）周圍的先天地理風水。

```
         日柱（南）
           ↑
月柱（東） ←┼→ 時柱（西）
           ↓
         年柱（北）
```

- **年柱** → 北方（包括正北、東北、西北）
- **月柱** → 東方（包括正東、東北、東南）
- **日柱** → 南方（包括正南、東南、西南）
- **時柱** → 西方（包括正西、西北、西南）

## 實作位置

- **檔案**：`Ecanapi/Controllers/ConsultationController.cs`
- **靜態方法**：`LfFengShuiPillarDesc(string pillar)` → 傳入干支（如「壬子」），回傳地理環境描述字串
- **呼叫位置**：`LfBuildYudongziReportV2()` 中的「第十六章：出生環境・先天地理風水」

## 注意事項

- 先天地理風水指**受孕當時所在地**周圍環境，通常等同父母當時住所
- 若父母在外地（旅遊、探親）受孕，則為該外地的地理環境
- 親子鑑定不符（生父非記載父親）時，此風水論斷需重新考證
- 驗證範圍：受孕地點幾米至幾百米內皆可計入
