# 命理鑑定框架（先問後算原則）

> 核心原則：**先查年齡 → 判斷婚姻狀態 → 決定分析框架 → 選定重點章節**
> 這是真正命理師拿到命盤的第一動作，也是與網路算命最大的差異。

---

## 第一步：依年齡判斷婚姻分析框架

| 年齡段 | 婚姻狀態假設 | 命書處理方式 |
|--------|------------|-------------|
| 20歲以內 | 一律視為未婚 | 最容易發揮，命書全算，所有主題都寫 |
| 21-30歲 | 未婚為主，但要判斷 | 先看八字夫妻星 + 紫微夫妻宮是否有婚機，決定「何時結婚/適合何類對象」方向 |
| 31-40歲 | 已婚為多，需確認 | 考慮是否已婚，已婚則重家庭婚姻子女；未婚則深探礙婚格局 |
| 41-50歲 | 可能離婚或再婚 | 婚姻穩定度、是否有二婚格局、子女成就 |
| 50歲以上 | 婚姻非重點 | 除非八字/紫微明確顯示婚姻異動，否則不以婚姻為重點 |

---

## 第二步：婚姻狀態 → 分析方向分叉

### 未婚者
- 是否有礙婚格局？（何時才有機會？）
- 適合何類型對象？（五行性格配對）
- 婚姻時機點在哪個大運流年？

### 已婚者
- 婚姻是否穩固？夫妻是否和諧？
- 是否有外遇/離婚跡象？
- 子女緣份、子女成就
- 另一半的事業/健康如何影響命主？

### 41-50歲特別注意
- 是否有二婚格局？
- 若離婚，時機點在哪？
- 再婚對象特質？

---

## 第三步：共同必查項目（任何年齡皆適用）

| 項目 | 八字依據 | 重要性 |
|------|---------|-------|
| 學歷高低 | 印星強弱、月柱 | 高 |
| 打工 vs 老闆 | 比劫重→創業傾向；官殺制身→適合受雇 | 高 |
| 公職 vs 民營 | 正官格→公家傾向；偏財、食傷→民營商業 | 高 |
| 身體傷疤 | 金水重、刑害 | 中 |
| 疾病隱患 | 衰弱五行對應臟腑 | 高 |
| 貴人在哪 | 用神五行對應方位、人物 | 高 |

---

## 第四步：年齡特殊加項

| 條件 | 額外分析 |
|------|---------|
| 45歲以上 | 考慮父母是否健在、父母健康 |
| 65歲以上 | 父母宮改為長輩宮，多不談父母 |
| 50歲以上有子女 | 子女宮改為子女孫輩宮，重子女成就 |
| 80歲以上 | 聚焦健康養生、財庫守護、子女孫輩 |

---

## 程式實作方向（未來版本）

### 現行做法（過濾式）
按固定章節輸出，遇年齡不符才跳過。

### 目標做法（框架驅動式）
1. 傳入 age → 選定分析框架
2. 框架決定哪些章節要寫、哪些不寫、哪些要強調
3. 婚姻狀態可由用戶輸入（未婚/已婚/離婚），或由命盤推算
4. 共同必查項目永遠輸出

### 用戶輸入建議
命書請求時，讓用戶可選填：
- 婚姻狀態：未婚 / 已婚 / 離婚或分居
- 有無子女：有 / 無
- 父母健在：皆健在 / 一方已故 / 皆已故

這三個欄位能讓命書從「推算」直接跳到「論斷」，品質大幅提升。
